What is CodespaceMPT?

CodespaceMPT is a coding playground that has the option to write code, save it as a .txt file, .py file, .js file, .html file, .css file, .ts file, .c file, .cs file, .cpp file, .java file or a .md file, and make a new repository in GitHub. It also has a built-in terminal with several commands and easter eggs. We have used AI to write us a simple base site and we fixed it by adding buttons and fixing things in the site. We also made an HTML site tester!
