What is CodespaceMPT?

CodespaceMPT is a coding playground that has the option to write code, save it as a .txt file or a .md file, and make a new repository in GitHub. It also has a built-in terminal with several commands and easter eggs. We have used AI to write us a simple base site and we fixed it by adding buttons and fixing things in the site.

This is version 1.11.1

How do I use it?

This code editor can be used in a simple way. Just write your code and once you are finished you can copy paste it to the box below the code editor, choose what kind of file you'll make, and hit save!
